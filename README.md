# TannerSlays Apex Legends Coaching Site

## Setup

1. Add your Apex rank badge images into the `images/` folder, named:
   - `bronze.png`
   - `silver.png`
   - `gold.png`
   - `platinum.png`
   - `diamond.png`
   - `masters.png`
   - `predator.png`

2. Replace `YOUR_CALENDLY_LINK` in `index.html` with your actual Calendly booking URL.

## Banner

- The header background uses `images/banner-bg.jpg` which is already included.
